```
 *注：“OfficePlus”在本聲明中統稱爲“本擴充功能”
```

本擴充功能是一個本地擴充功能
- **個人數據搜集**：本擴充功能不會搜集、存取或著傳輸任何個人數據。
- **Office 數據**：本擴充功能只讀取Office最近文件數據以提供檔案搜索服務。
- **網路存取**：本擴充功能不會使用任何網路服務。
