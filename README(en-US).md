## Introduction
SuperSpace is an extension for PowerToys Command Palette designed to facilitate launching Microsoft Office applications and creating new Documents(Word), Workbooks(Excel) and Presentations(PowerPoint).

## Privacy Policy
```
 *Note：“SuperSpace”is referred to as “it” throughout this statement
```
It is a purely local utility
- **Personal Data Collection**:It does not collect, access or transmit any personal data.
- **Office Data**:It only reads local Office "Recent Files" metabase to provide file search services.
- **Network Access**: It does not utilize any network services or internet connectivity.
